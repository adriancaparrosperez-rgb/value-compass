# Correspondencia con el Prompt Maestro

## Implementado en esta versión

- Modalidad de universo amplio mediante precarga colectiva.
- Universo completo editable, sin selección automática de finalistas.
- Scores separados: valoración, calidad, caja, balance, crecimiento, capital allocation, momentum fundamental, riesgo y confianza del dato.
- Recomendación preliminar condicionada por valoración, balance y completitud.
- Persistencia por ejecución y exportación CSV, Excel y JSON.
- Versionado temporal mediante `run_id` UTC.
- Ficha individual.
- DCF configurable.
- Arquitectura de proveedores intercambiables.
- Automatización diaria con GitHub Actions.

## Diseñado para la siguiente ampliación

- Tres ejercicios históricos por separado.
- Fuente y trazabilidad por cada métrica material.
- Normalización de beneficio, FCF y owner earnings.
- ROIC y ROIC incremental.
- Escenarios conservador, base y optimista.
- Percentiles históricos y comparables sectoriales.
- Modelos sectoriales para bancos, aseguradoras, REIT, utilities y commodities.
- Moat, management, remuneración y asignación de capital cualitativa.
- Alertas de profit warning, cambios de guidance y revisiones.
- Cartera personal y decisiones diferenciadas para inversor nuevo y ya invertido.
- Tramos de entrada, añadir, recorte y venta.
- Base maestra completa con hojas del Prompt Maestro.

## Regla operativa

El radar diario no sustituye el análisis completo. Su misión es ordenar el universo, identificar cambios y señalar qué empresas merecen revisión humana. Las recomendaciones definitivas requieren validación de fuentes oficiales y normalización sectorial.
